# Boolean

#### Rescript
```reasonml
true, false /* boolean value */
&& || !     /* logical and, or, not */
<= >= < >   /* structural comparison */
==, !=      /* structural equality/inequality */
(1,1) == (1,1)  /* true */
(1,1) != (1,1) /* false */

===, !==    /* referential equality/inequality */
(1,1) === (1,1)  /* false */
(1,1) !== (1,1) /* true */
```

#### OCaml
```OCaml
true, false   (* boolean value *)
&& || not     (* logical and, or, not *)
<= >= < >     (* structural comparison *)

=, <>         (* structural equality/inequality *)
(1,1) = (1,1)  (* true *)
(1,1) <> (1,1) (* false *)

==, !=    (* referential equality/inequality *)
(1,1) == (1,1)  (* false *)
(1,1) != (1,1)  (* true *)
```
